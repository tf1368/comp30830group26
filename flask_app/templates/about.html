Welcome to Super Dublin!

On this website, there are 3 pages. 1.Index page: This page contains all the stations on map. You can filter stations based on availability, the Markers are colored based on the availability of bikes. Also, you can check weather info to see if it's a good time to bicycling. 2.About page: This page contains information about how we met all the requirements. 3.Find the Nearest station page: This page contains information about the nearest station, the nearest station is calculated based on the user's location.

Final project for COMP30830 Software Engineering module, this application sees to all of the needs of the modern bike commuter. Meeting requirements such as:
requirement How we meet the requirement
JCDecaux Data Collects data through JCDecaux API
Data Managment data management/storage in RDS DB on AWS
display bike stations on map Map style and station markers are customised into Super Mario style
flask web application (python) Use flask framework for the backend
Station Occupancy information Click on each station, there will be a popup info window
weather information current weather is just above the map
Interactivity (click, API request, handle response) click on Station filter, station input for prediction and stations on the map, proper response will happen
Machine Learning Models Random forest model for predicting occupancy based on weather patterns, trained on collected data
EC2 instance project served on a named host on the EC2 instance (made a mistake of choosing a US host instead of Dublin, so the delay is quite massive)
